Installation instructions:

1. Install cordova

    sudo npm install -g cordova
    sudo npm install -g ios-dev

2. Setup cordova project

    ./setup-phonegap.sh

3. Build app

    sencha app build testing
    ./build-phonegap.sh
    cordova run ios

Cordova version ---> 5.4.0
Sencha cmd version ---> 6.0.2.14
