Installation instructions:
1. Create phonegap app
phonegap create pg-tinktime "com.tinktime.app" "Tinktime"

2. install platforms
phonegap platform add ios

3. install plugins
1. com.cordova.plugins.sms
GIT Repo: - https://github.com/cordova-sms/cordova-sms-plugin

2. com.hiddentao.cordova.filepath
GIT Repo: - https://github.com/hiddentao/cordova-plugin-filepath

3. com.phonegap.plugins.PushPlugin
GIT Repo: - https://github.com/phonegap/phonegap-plugin-push

4. Standard cordova plugins
(a) cordova-plugin-camera
(b) cordova-plugin-console
(c) cordova-plugin-contacts
(d) cordova-plugin-device
(e) cordova-plugin-file
(f) cordova-plugin-file-transfer
(g) cordova-plugin-sms

Phonegap version ---> 5.3.6
Sencha cmd version ---> 6.0.2.14
