Installation instructions:

1. Install cordova

    sudo npm install -g cordova
    sudo npm install -g ios-dev

2. Setup cordova project

    ./setup-phonegap.sh

3. Build app

    ./build-phonegap.sh
    cd pg-tinktime
    cordova run ios
    cordova run android

Cordova version ---> 5.4.0
Sencha cmd version ---> 6.0.2.14
